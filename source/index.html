Project Boilerplate

Dead simple boilerplate covers all your side-project needs. All the Grunt tooling you'll ever need, plus:

  • Acetate
    A static site generator that only concerns itself with your content and layouts.
  • TypeCabinet
    A sass library that brings old-school typographic theory to new-school applications.
  • Viewport Grid
    A Sass library that scaffolds out a grid system heavily influenced by graphic design techniques that predate your grandpa.
  • Rye
    A set of useful mixins that let you through out Compass and ignore all the weird stuff in Bourbon.

Get it on Github.
